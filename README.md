# Weitzly.com

Simple resume site for me.

## Dependencies

[NextJS](https://nextjs.org/)
[Styletron](https://github.com/rtsao/styletron).

## Install

clone the repo and run

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## How to use

[NextJS with Styletron](https://www.styletron.org/getting-started/#with-nextjs)
